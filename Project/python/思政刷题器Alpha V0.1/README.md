# 思政刷刷题器使用说明书
## 下载说明
直接点code下载源码就行
## 使用说明
1. 下载 python3 ， 把python添加到环境变量(Add Python to PATH)

2. win+R 输入cmd 。依次输入
pip install pandas 
pip install tkinter
pip install sys
安装失败的浏览器搜**pip换源**

3. 运行project.py

4. 做对的题目删除，做错的题目下次启动程序会再做一遍

5. 重置题库请把data(副本).csv 重命名(F2)为 data.csv

6. 有影响使用的bug请告诉我

7. 使用vscode请选择**打开文件夹**方式启动
